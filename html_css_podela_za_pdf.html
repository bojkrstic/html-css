HTML levo, CSS desno po sekcijama

Svaki red prikazuje HTML deo stranice levo i CSS koji na njega utiče desno. Ovo nije 1:1 red po red, nego praktična mapa po blokovima.

HTML: head i povezivanje CSS: globalni stil i wrapper
HTML
<head>
  <!-- Meta podaci i povezivanje CSS fajla -->
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>JS Bin</title>
  <link rel="stylesheet" href="css2.css">
</head>
CSS
* { /* Selektor za sve elemente na stranici */
  box-sizing: border-box; /* Sirina i visina ukljucuju padding i border */
}

body { /* Osnovni stil cele stranice */
  margin: 0; /* Uklanja podrazumevani razmak oko strane */
  font-family: Arial, sans-serif; /* Postavlja osnovni font */
  background-color: #e3f6f5; /* Svetla pozadina cele stranice */
  color: #1f2933; /* Osnovna boja teksta */
}

.strana { /* Glavni wrapper za sav sadrzaj */
  max-width: 1280px; /* Ogranicava sirinu stranice */
  margin: 0 auto; /* Centira kontejner horizontalno */
  padding: 20px; /* Unutrasnji razmak od ivica */
}

nav { /* Gornja navigaciona traka */
  display: flex; /* Raspored elemenata u jednom redu */
  justify-content: space-between; /* Levi i desni deo razdvaja na krajeve */
  align-items: center; /* Vertikalno poravnava elemente po sredini */
  gap: 20px; /* Razmak izmedju levog i desnog bloka */
  padding: 14px 18px; /* Unutrasnji razmak u navigaciji */
  border-radius: 10px; /* Zaobljene ivice navigacije */

  box-shadow: 0 0 10px #dbd8e3; /* Blaga senka oko navigacije */
}

.navigacija-levo { /* Levi meni u navigaciji */
  display: flex; /* Linkovi idu u jedan red */
  flex-wrap: wrap; /* Ako nema mesta, prelaze u novi red */
  gap: 10px 18px; /* Razmak izmedju stavki */
  list-style: none; /* Uklanja tackice iz liste */
  padding: 0; /* Uklanja unutrasnji razmak liste */
  margin: 0; /* Uklanja spoljasnji razmak liste */
}

.navigacija-levo li { /* Svaka stavka u meniju */
  display: inline-block; /* Stavke se ponasaju kao blokovi u redu */
}

.navigacija-desno { /* Desni deo navigacije */
  white-space: nowrap; /* Ne dozvoljava prelom teksta u vise redova */
}

a.navlink { /* Svi linkovi sa klasom navlink */
  text-decoration: none; /* Uklanja podvucenost */
  color: #1f2933; /* Tamna boja linka */
  transition: color 0.2s ease, border-color 0.2s ease; /* Glatka promena boje */
}

a.navlink:hover { /* Kada se mis prevede preko linka */
  color: #155263; /* Menja boju linka */
  border-bottom: 1px solid #155263; /* Dodaje donju liniju */
}

HTML
<div class="kursevi">

  <!-- Prvi kurs -->
  <div class="kurs">...</div>

  <!-- Drugi kurs sa slikom -->
  <div class="kurs srednji">...<img class="srednja-slika">...</div>

  <!-- Treci kurs -->
  <div class="kurs">...</div>
</div>
CSS
.kursevi { /* Sekcija sa karticama kurseva */
  margin-top: 20px; /* Razmak od bannera */
}

.kursevi h2 { /* Naslov sekcije kurseva */
  margin: 0 0 16px; /* Uklanja gornji razmak i ostavlja donji */
}

.kursevi::after { /* Clearfix za float kartice */
  content: ""; /* Pseudo-element bez sadrzaja */
  display: block; /* Ponaša se kao blok */
  clear: both; /* Cisti float elemente ispod */
}

.kurs { /* Jedna kartica kursa */
  float: left; /* Kartice idu jedna pored druge */
  width: calc(33.333% - 14px); /* Otprilike trecina sirine uz korekciju razmaka */
  min-height: 320px; /* Minimalna visina kartice */
  margin-right: 20px; /* Razmak izmedju kartica */
  margin-bottom: 20px; /* Donji razmak */
  padding: 18px; /* Unutrasnji razmak */
  border-radius: 10px; /* Zaobljene ivice */
  background-color: #5c5470; /* Osnovna boja kartice */
  color: #fff; /* Bela boja teksta */
  box-shadow: 4px 4px 4px #dbd8e3; /* Senka ispod kartice */
  transition: transform 0.2s ease, background-color 0.2s ease; /* Glatka promena na hover */

.kurs:last-child { /* Poslednja kartica u nizu */
  margin-right: 0; /* Uklanja desni razmak na poslednjoj kartici */

HTML: responsive CSS: media queries
HTML
HTML ne menja sadržaj za responsive deo.
Responsive ponašanje dolazi samo iz CSS-a.
CSS
@media (max-width: 960px) { /* Pravila za tablet i uži ekran */
  .sadrzaj { /* Glavni raspored */
    flex-direction: column; /* Kolone idu jedna ispod druge */
  }

  .desna-strana { /* Sidebar na užem ekranu */
    flex-basis: auto; /* Uklanja fiksno ponašanje širine */
    min-width: 0; /* Omogućava skupljanje */
  }

  .kurs { /* Kartice kurseva na užem ekranu */
    width: 100%; /* Svaka kartica ide na punu širinu */
    margin-right: 0; /* Uklanja desni razmak */
  }
}

@media (max-width: 640px) { /* Pravila za mali ekran */
  .strana { /* Glavni wrapper */
    padding: 12px; /* Manji padding na telefonu */
  }

  nav { /* Navigacija */
    flex-direction: column; /* Linkovi idu vertikalno */
    align-items: flex-start; /* Poravnanje ulevo */
  }

  .baner-naslov { /* Naslov u banneru */
    padding-top: 70px; /* Malo manji razmak na malom ekranu */
    font-size: 24px; /* Manji font za telefon */
  }
}
